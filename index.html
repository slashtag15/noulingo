Learn Morse Code

Morse code uses dots (.) and dashes (-) to represent letters and numbers. Example: A = .- , B = -...

Basic Rules:

  • Dot = short beep
  • Dash = long beep
  • 1 space between parts of a letter
  • 3 spaces between letters
